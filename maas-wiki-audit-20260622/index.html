  • 活跃度极低：近 90 天仅 9% 文档有更新(06-16 数据)，即 91% 的文档长期无人维护
  • 权限过度集中：知识库成员仅 1 人(创建者/管理员)，无其他维护者；本次评测因账号非成员导致 API 无法遍历全量
  • 无定期复核机制：限时促销、模型矩阵、合规资质等强时效内容无复核周期，过期后长期挂着
  • 无准入/退出标准：废弃文档(自述 Deprecated)、明文密码、OA 行政手册均能进入并长期留存

🔴 安全与合规告警 P0 立即处置

1. 明文密码泄露 —「【翻译场景】合成数据SFT PRD」 内容审计发现该 PRD 草稿(2024 Q3)中含明文泄露凭据(密码)。知识库若对外开放或成员较多，属直接凭证泄露。立即删除明文密码并轮换相关凭据。
2. 对外合规材料过期 —「客户隐私数据保护方案（外部）」 版本表/认证进度/等保 ISO 计划全部停在 2024-01~04，所有「预计完成」节点已过 2 年。作为对外可信材料，当前状态可能不反映真实合规资质，存在对外误导与合规风险。建议安全/合规团队基于当前资质重写。
3. 已弃用服务文档留存 —「2023.11 ChatGPT API 服务文档」 标题自带【Deprecated】，内容为已停用的 ChatGPT 代理服务(one-api.glm.ai)，含 $200 初始额度等失效定价。若知识库对客户可见，会暴露已下线的内部服务入口。
