合作切入点（与 Z.ai / 智谱）

最高优先：YTL

YTL 正构建主权 AI 全栈——但 ILMU 模型能力尚在早期。GLM-5.1 可作为 ILMU 的能力补充（推理/coding/multimodal），尤其适合 YTL AI Cloud 面向企业客户的 API 转售场景。首步建议：以 GLM-5.1 API 为 ILMU Claw Agent 平台提供推理/coding 能力补充的 90 天 POC。

高优先：CP Group

CP 的 AI 覆盖农业/零售/物流，无自研模型，依赖 Microsoft。GLM 可在非美生态场景（东南亚本地化、RolePlay、Function Call）形成差异化。Amity Group 是天然入口。建议通过 Amity 先建立技术对话。

中优先：SK / Foxconn / Ayala

SK/Foxconn 偏基础设施层，短期合作空间有限。Ayala 数据中心建成后需填充 AI 能力，可跟进。Jardines 通过 AI 医疗切入，可探索 GLM-5v 多模态在放射诊断场景的辅助能力。
