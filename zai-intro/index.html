API 核心能力

01

OpenAI 兼容

完全兼容 OpenAI API 格式，零迁移成本接入。支持 Streaming、Function Call、JSON Mode。

02

Function Calling

业界领先的工具调用能力。GLM-4.5 工具调用成功率 90.6%，超越 Claude-4-Sonnet。

03

Prompt Cache

自动 Prompt 缓存，缓存命中降低 80% 输入成本。5 分钟 TTL，高频场景成本最优。

04

Batch API

大批量离线处理，价格低至标准 API 的 50%。适合数据分析、内容生成等批量场景。

05

多模态输入

支持图片、文档、视频输入。GLM-5V-Turbo 单次处理 100+ 页文档，300 页 PPT。

06

全球部署

国内 (bigmodel.cn) + 国际 (z.ai) 双平台。华为昇腾私有化部署方案可选。
