架构创新

五大核心技术

不只是参数规模扩大 — 注意力机制、优化器、训练管线全面重构

1. 压缩稀疏注意力 (CSA)

沿序列维度 4x 压缩 KV Cache，稀疏选择 Top-1024 + 128 token 滑动窗口，精确检索长上下文关键信息。

2. 高压缩注意力 (HCA)

128x 激进压缩后全连接，每层获得全局视野。与 CSA 交替排列，兼顾精确与广角。

3. 流形约束超连接 (mHC)

用 Sinkhorn-Knopp 算法约束残差连接到 Birkhoff Polytope，防止万亿参数下信号坍缩。

4. Muon 优化器

替代 AdamW，在 33T token 数据集上实现更快收敛和更稳定训练。Peak LR: 2.0e-4。

5. FP4 混合精度训练

MoE 专家权重和 Indexer QK 路径使用 FP4 量化感知训练，其余 FP8，兼顾效率和质量。

效率提升：尽管 V4-Pro 参数量是 V3.2 的 2.4 倍，但在 1M 上下文场景下，仅需 27% 的推理 FLOPs 和 10% 的 KV Cache，使 100 万 token 上下文从理论变成工程可行。
