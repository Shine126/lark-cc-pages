BIS Entity List 50% Rule 对智谱合作的影响分析

一、背景：两条规则叠加

规则生效时间核心内容
Entity List 列名2025年1月16日北京智谱华章（现智谱华章股份）被列入实体清单，需 BIS 许可才能获取受 EAR 管制的美国原产技术/产品
50% Affiliates Rule2025年9月29日生效 → 11月10日暂停一年实体清单上的主体直接或间接控股 ≥50% 的境外子公司/关联公司，自动受同等出口管制限制

50% Rule 目前处于暂停期（至 2026年11月9日），届时将自动恢复（snap-back），除非 BIS 另行行动。

五、结论

> 1. Entity List 列名本身已对智谱与美国技术供应商的合作构成实质性限制（GPU、云服务、美企技术合作）

> 2. 50% Rule 是"补刀"机制，目标是堵住"通过子公司/壳公司绕过管制"的漏洞，对智谱的海外子公司和投资主体产生传导效应

> 3. 当前 50% Rule 处于暂停期（至 2026.11.09），但 5 个月后自动恢复，合作方需提前做好合规准备

> 4. 智谱的应对策略已明确：国产算力替代 + 全链路自研 + 依托国内生态，海外 API 业务在暂停期内仍有窗口期
